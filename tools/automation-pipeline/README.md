# automation-pipeline｜全工程一気通貫自動化

> ステータス：**未開発（Phase 3後半で着手）**

---

## 概要

案件発見から納品・請求書発行まで全工程を自動化するパイプライン。

## フロー

```
[自動] 案件発見（job-scout）
    ↓
[自動] スコアリング・フィルタリング
    ↓
[自動] 提案文・デモ生成（proposal-generator）
    ↓
[人間] 確認・送信
    ↓
[人間] 受注確認・要件ヒアリング
    ↓
[自動] 本実装・手順書生成（delivery-generator）
    ↓
[人間] 最終確認
    ↓
[自動] 納品・ログ記録・請求書発行
```
