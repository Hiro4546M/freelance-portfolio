# 手順書 04｜n8n ローカル起動 + Notion セットアップ

**所要時間**: 約30〜40分
**目的**: auto-001 ポートフォリオサンプルを動かすための環境準備

---

## 前提条件

- Node.js がインストール済み（`node --version` で確認）
- Google アカウント（Gmail 用）
- Anthropic API キー（既存）

Node.js のバージョンは 18 以上を推奨。未インストールの場合は https://nodejs.org からインストールする。

## Step 1: n8n の起動

ターミナルで以下を実行する。

```bash
npx n8n
```

**初回起動時の注意**:
- npm パッケージのダウンロードに数分かかる場合がある
- `n8n ready on port 5678` と表示されれば起動成功
- ターミナルは閉じないこと（n8n が終了する）

**ブラウザでアクセス**:

http://localhost:5678 を開く。

**初回アカウント設定**:
1. メールアドレスとパスワードを入力してアカウントを作成する（ローカル専用アカウント）
2. 「Skip」を選択してよい設定項目は飛ばして構わない
3. ダッシュボード画面が表示されれば準備完了

## Step 2: Notion アカウント・データベースの作成

### 2-1. Notion アカウントの作成

https://notion.so にアクセスし、メールアドレスまたは Google アカウントでサインアップする。
すでにアカウントを持っている場合はログインする。

### 2-2. 新規ページとデータベースの作成

1. 左サイドバーの「+ 新規ページ」をクリックする
2. ページタイトルを「メール要約」と入力する
3. ページ本文にカーソルを置き、`/database` と入力する
4. 「インラインデータベース」を選択する（ページ内に表示されるタイプ）

### 2-3. プロパティの設定

デフォルトで「名前」（タイトル型）が作成されている。以下のプロパティを追加する。

| プロパティ名 | 型 |
|---|---|
| 名前（タイトル） | タイトル型（デフォルトのまま） |
| 送信者 | テキスト型 |
| 要約 | テキスト型 |
| 受信日時 | 日付型 |

**プロパティの追加方法**:
1. データベースの右端にある「+」をクリックする
2. プロパティ名を入力し、型を選択する
3. 同様に繰り返して4つすべてを作成する

## Step 3: データベースIDの取得方法

Notion データベースのIDは URL から取得する。

**URL の例**:
```
https://notion.so/myworkspace/メール要約-abc123def456ghij7890klmn12345678?v=xyz...
```

**ID の場所**:
- ページタイトル（「メール要約-」）の直後から始まる32文字の英数字がデータベースID
- `?v=` の前の部分が該当する
- 例: `abc123def456ghij7890klmn12345678`

**確認方法**:
1. Notion でデータベースのページを開く
2. ブラウザのアドレスバーのURLをコピーする
3. ハイフン区切りの英数字部分を取り出す（32文字）

**補足**: URLに `notion.so/` の後にワークスペース名がない場合は、`notion.so/` の直後の32文字を使う。

## Step 4: Notion インテグレーションの作成

n8n から Notion を操作するために、インテグレーション（APIキー）が必要。

### 4-1. インテグレーションの作成

1. https://www.notion.so/my-integrations にアクセスする
2. 「+ New integration」ボタンをクリックする
3. 以下を入力する:
   - **Name**: `n8n-email-summary`（任意の名前）
   - **Associated workspace**: 使用しているワークスペースを選択
4. **Capabilities（権限）の設定**:
   - `Read content` にチェック
   - `Insert content` にチェック
   - `Update content` にチェック
   - `Read user information` は不要（チェック不要）
5. 「Submit」をクリックして作成する

### 4-2. シークレットキーのコピー

インテグレーション作成後、`Internal Integration Secret` が表示される。

1. 「Show」ボタンをクリックしてシークレットを表示する
2. `secret_` から始まる文字列全体をコピーする
3. **Step 5のメモシートに記録する**（この画面を閉じると再表示できないため）

### 4-3. データベースへのインテグレーション接続

作成したインテグレーションをデータベースに接続しないと、n8n からアクセスできない。

1. Notion の「メール要約」データベースページを開く
2. ページ右上の「...」（3点メニュー）をクリックする
3. 「Connections」（または「接続」）をクリックする
4. 作成したインテグレーション名（`n8n-email-summary`）を検索・選択する
5. 「Confirm」をクリックする

接続済みになるとインテグレーション名がリストに表示される。
