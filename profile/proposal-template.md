# 提案文テンプレート（技術系・v3）

---

## 基本構成（全案件共通）

```
①  課題の言い換え（1行）
②  対応方針（具体的に）
③  デモ or ポートフォリオの提示
④  見積もり・納期
⑤  確認事項（不安を取り除く）
```

---

## テンプレA｜デモあり版（推奨）

```
ご依頼内容を拝見しました。
〇〇（課題の要約）を自動化するご要望ですね。

要件を読んで、コアの動作を試しに実装しました。
▼ デモ（GitHubリンク or Loom動画）
https://github.com/...

本実装では〇〇と△△を追加して完成させます。

【対応内容】
・使用ツール：n8n / Make / Claude API（要件に応じて選定）
・納品物：ワークフロー本体 ＋ 操作手順書
・納期：要件確認後から5営業日以内

【お見積もり】
〇〇円（税込・CrowdWorks手数料込）

仕様で不明な点があれば着手前に確認します。
よろしくお願いいたします。
```
