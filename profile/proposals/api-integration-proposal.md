# 提案文サンプル｜汎用API連携・業務自動化（ヒアリング重視）

対象案件：要件がやや曖昧な「何かを自動化したい」「システム間を連携したい」系の案件全般
想定単価：3〜8万円
使用テンプレート：テンプレートA（デモあり版）をベースに、ヒアリング重視のスタイルにアレンジ
特徴：確認事項を多めに置いて「一緒に要件を整理する姿勢」を前面に出す

---

## 提案文（コピー&貼り付け用）

---

ご依頼内容を拝見しました。
複数のツールやシステムをつなげて、手作業を減らしたいというご要望ですね。

まず、要件の整理からお手伝いします。
「なんとなく自動化したいが、どこから手をつければいいかわからない」という段階でも問題ありません。現状の業務フローをヒアリングして、優先度の高い部分から着実に自動化します。

▼ ポートフォリオ（API連携・業務自動化のサンプルがあります）
https://github.com/Hiro4546M/freelance-portfolio

【対応範囲（例）】
・ツール間のデータ連携（Notion ↔ スプレッドシート ↔ Slack 等）
・外部APIの呼び出し・レスポンス処理
・定期実行バッチ（毎朝9時にデータ取得→集計→通知、等）
・Webhook受信して別ツールに転送
・Make / n8n / Zapier / Python スクリプト（要件に応じて最適なものを選定）

【進め方】
1. ヒアリング（30分、無料）：現状フローと理想の状態を確認
2. 実装方針の提案（要件確定後1〜2日）：使用ツール・コスト・工数を提示
3. 実装・テスト（合意後5〜10営業日）
4. 納品：動作確認済みのワークフロー ＋ 操作手順書

【お見積もり目安】
・単純な2ツール連携：29,800〜49,800円
・条件分岐・複数ツール：49,800〜79,800円
（詳細はヒアリング後に確定します）

まず以下を教えてください。
・現在お使いのツール・サービス名（Slack、Notion、Gmail、kintone等）
・自動化したい作業を一言で表すと？（例：フォーム回答をSlackに通知したい）
・ご希望の着手時期はいつ頃ですか？

ご状況に合わせて柔軟に対応します。お気軽にご連絡ください。

## カスタマイズのポイント

- **要件が明確な案件の場合**：「まず要件の整理から」の部分を削除し、具体的な実装方針を冒頭に持ってくる（`make-form-proposal.md` や `n8n-automation-proposal.md` のスタイルに切り替える）
- **kintone / Salesforce 案件**：「対応範囲（例）」に「kintone REST API連携」「Salesforce Webhook対応」を追加する
- **LINE連携案件**：「Webhook受信して別ツールに転送」の後ろに「LINE公式アカウントへの自動返信・通知」を追加する
- **Python指定の案件**：「Make / n8n / Zapier / Python スクリプト」の Python を先頭に持ってくる
- **単価が低い案件**：ヒアリングで要件が明確になった時点で「この規模なら2〜3万円で対応できます」とスコープを絞って提示し、受注しやすい形に誘導する

## ヒアリング時に確認するチェックリスト

ヒアリング（テキストチャットで可）で以下を確認する。

### 現状の業務フロー
- [ ] 現在どんなツールを使っているか（サービス名を具体的に）
- [ ] 手動でやっている作業の頻度（毎日・週次・月次）
- [ ] 担当者は何人いるか

### 自動化の理想状態
- [ ] 自動化後、担当者は何をするだけでよいか
- [ ] 通知・記録の形式はどうしたいか（Slack通知文のフォーマット等）
- [ ] エラー時はどう対応したいか

### 技術・環境
- [ ] 既存システムのAPI利用可否（APIキーがあるか）
- [ ] 既存ツールの契約プラン（無料プランはWebhookが使えない場合がある）
- [ ] 自分でシナリオを編集できる人がいるか（保守体制の確認）
