ご依頼内容を拝見しました。
毎日届くメールや問い合わせを手作業で確認・振り分けしている手間をなくしたいというご要望ですね。

n8n を使ったワークフローのサンプルを組んであります。
▼ ポートフォリオ（メール自動化・AI連携のサンプルがあります）
https://github.com/Hiro4546M/freelance-portfolio

今回のポイントは「単純な振り分けで終わらせない」ことです。
メールの内容をAI（Claude API）に渡して、カテゴリ分類・重要度判定・返信下書き生成まで自動で行えるようにします。担当者の作業を「確認してGoを出すだけ」にする構成です。

【対応内容】
・使用ツール：n8n（セルフホスト or Cloud）＋ Claude API
・処理フロー：メール受信 → AI分類・要約 → Slack通知（担当者別）→ 返信下書きをGmailに保存
・納品物：n8nワークフロー（JSONエクスポート）＋ API設定手順書 ＋ 操作マニュアル
・納期：要件確認後から7営業日以内

【お見積もり】
79,800円（税込・CrowdWorks手数料込）

※ n8n Cloud を新規で使う場合のアカウント費用は別途クライアント負担となります（月額約20ドル〜）。

着手前に確認させてください。

## カスタマイズのポイント

- **AI連携が不要な場合**：Claude APIの部分を省いて「単純なメール振り分け・転送ワークフロー」に絞り込み、49,800円に調整する
- **Outlook対応の場合**：n8n の Microsoft 365 ノードで対応可能と追記する
- **大量メール（日100件以上）の場合**：「処理件数に応じてAPI費用が変動するため、月間のメール件数を教えてください」を確認事項に追加する
- **問い合わせフォーム案件の場合**：「メール受信」を「フォーム送信」に置き換えて構成を説明する
- **ChatGPT指定の場合**：Claude APIをOpenAI APIに置き換えて同様に対応できる旨を記載する。「どちらも対応できるため、ご希望があればお聞かせください」と一言添える
