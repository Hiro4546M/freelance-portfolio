# セットアップガイド｜Googleフォーム→Slack通知＋スプレッドシート

## 必要なもの
- Makeアカウント（無料）: https://make.com
- Googleアカウント（Forms・Sheets）
- Slackワークスペース

## 手順（約30分）

### 1. Googleフォームを作成
1. forms.google.com で新規フォーム作成
2. 以下の順序でフィールドを追加（順序が重要）：
   - 質問1：お名前（短文）
   - 質問2：メールアドレス（短文）
   - 質問3：お問い合わせ内容（長文）
3. フォームIDをメモ（URLの `/d/XXXX/` 部分）

### 2. Googleスプレッドシートを作成
1. sheets.google.com で新規シート作成
2. 1行目にヘッダーを入力：`名前` / `メール` / `内容` / `受信日時`
3. スプレッドシートIDをメモ（URLの `/d/XXXX/` 部分）

### 3. Slackチャンネルを用意
1. 通知を受け取るチャンネルを作成（例：#通知）
2. チャンネル名をメモ

### 4. Makeにシナリオをインポート
1. make.com にログイン
2. 「Create a new scenario」
3. 右上「...」→「Import Blueprint」
4. `workflow-blueprint.json` をアップロード

### 5. 認証情報を設定
1. Google Formsモジュール：Googleアカウントでログイン → フォームIDを入力
2. Google Sheetsモジュール：同じアカウントを選択 → スプレッドシートIDとシート名「回答一覧」を入力
3. Slackモジュール：ワークスペースを接続 → チャンネル名を入力

### 6. テスト実行 → 有効化
1. 「Run once」でテスト実行
2. フォームにテスト回答を送信
3. Slackに通知・スプレッドシートに記録されたことを確認
4. 動作確認後「Scheduling」をONに設定

## トラブルシューティング
- **Slackに通知が来ない**: Slackモジュールの認証を再接続してください
- **スプレッドシートに記録されない**: シート名が「回答一覧」と一致しているか確認してください
- **フォームIDが見つからない**: フォームのURLの `/d/` と `/edit` の間の文字列です
